# app-imc

Este é um aplicativo de cálculo de IMC.
Onde é possível adicionar o peso e a altura de uma pessoa, e fazer o cálculo, mostrando o resultado para o usuário na tela.


 ### Usei e conheci: 

- GestureDetector
onTap, onTapDown, onTapUp

- MediaQuery

- CupertinoButton

- Google Fonts

- SafeArea

- EdgeInsets.only

- image.asset

- InkWell
